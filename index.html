WHAT IS SUPERPONG?

Real-Time AI Umpire for Table Tennis

SuperPong is a real-time AI umpire built using computer vision, multimodal AI, and voice interaction. It detects ball movement, determines rally outcomes, updates the scoreboard, and delivers live commentary — all hands-free, running entirely in your browser.

Computer Vision Ball Detection

YOLO tracks ball movement at 5fps, detecting rally outcomes and score events in real time.

Voice-First Interaction

Wake word activation, natural language commands, undo support — all without touching a device.

Live AI Commentary

Gemini delivers energetic real-time voice commentary after every rally. Point for point.

Auto Scoreboard Updates

Score updates in under 1 second. Serve rotation, deuce rules, and best-of sets — fully ITTF compliant.

Runs in the Browser

No installs. Full WebRTC camera + mic stream. Optional Python backend with YOLO for enhanced accuracy.

5fps Video Processing

HOW IT WORKS

From Setup to Scoreboard in Seconds

Three simple steps. Voice-activated. Fully automatic.

01

Setup Match

Enter player names, choose your format (11, 21, or best-of sets), and hit start.

02

Wake the AI

Say "SuperPong" to activate. Camera and mic start simultaneously. The AI begins listening.

03

Live Scoring

YOLO tracks the ball at 5fps. Gemini detects rally loss, updates score, and delivers live commentary.

CORE FEATURES

Everything You Need. Nothing You Don't.

Voice-First Scoring

Natural language control with undo and override support. Context-aware — no button presses ever.

YOLO Ball Detection

Ultralytics YOLO with custom VideoProcessor. Trajectory prediction and auto rally outcome detection.

Live Gemini Commentary

Real-time voice output with energetic short commentary after every rally point.

Full Match Intelligence

Serve rotation, deuce rules, best-of sets. Fully ITTF compliant scoring logic.

Backend Optional Mode

Full WebRTC + Python backend, or browser-only Gemini fallback. Your call.

Post-Match Analytics

Winner display, point percentage, set breakdown, and full match duration summary.

UNDER THE HOOD

Built on Serious Technology

A modular architecture that works in the browser or scales with a Python backend.

Frontend Stack

React + Vite · Web Speech API wake word · WebRTC camera stream · useAgentConnection hook

Backend Mode

Stream WebRTC transport · Vision Agents backend · Gemini Realtime at 5fps · YOLO BallTrackingProcessor

Fallback Mode

Direct Gemini Live in browser · Silent SYSTEM STATE UPDATE messages · Keeps score context accurate

Vision Agents SDK

The framework powering SuperPong. Handles WebRTC, Gemini sessions, and custom video processor integration.
